# 🌌 Cosmic Overhaul

*The ultimate Quality of Life and Galaxy Simulation overhaul for Avorion.*

## 📖 Overview
Cosmic Overhaul introduces several QoL improvements to Avorion, including an advanced Bulletin Board interface for filtering missions, new map tools, dynamic respawning of key sectors, and enhanced background simulation logic. The universe is more responsive to your actions, with dedicated tools for managing your empire seamlessly.

## ✨ Features
<details>
<summary><b>Click to expand features</b></summary>

- **Bulletin Board QoL:** Advanced mission filtering and sorting.
- **Dynamic Faction Wars:** Factions organically fight, claim bounties, and alter reputation.
- **Sector Keep-Alive:** Essential sectors dynamically respawn to keep the galaxy feeling alive.
- **Trash Management:** Intelligent deletion of junk to keep server TPS high.
</details>

## ⚙️ Requirements
- Avorion v2.0+
- Dependencies: **Cosmic Vault**

## 🚀 Installation
1. Place the folder in:
   - **Windows:** `%AppData%\Avorion\mods\`
   - **Linux:** `~/.avorion/mods/`
2. Enable **Cosmic Overhaul** in **Settings -> Mods**.
3. Restart Avorion when prompted.
