# Cosmic Overhaul

[![License](https://img.shields.io/badge/License-Apache_2.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)

**Cosmic Overhaul** is a large-scale Avorion systems and quality-of-life overhaul in the Cosmic series.

It is designed for players who want:
- Smoother long-session gameplay flow.
- Stronger economy/logistics usability.
- Richer captain command behavior.
- Less day-to-day friction in fleet management.

## Quick Highlights

- Broad QoL and systems improvements across economy, stations, UI, and command workflows.
- Major map-command simulation enhancements for trade, refine, scout, and related operations.
- Integrated module approach intended to reduce mod-stack friction for common overhaul features.

## Installation

1. Place the mod folder into:
   - **Windows:** `%AppData%\Avorion\mods\`
   - **Linux:** `~/.avorion/mods/`
2. Start Avorion.
3. Enable **Cosmic Overhaul** in **Settings -> Mods**.
4. Restart when prompted.

---

## Compatibility Snapshot

- `saveGameAltering = true`
- `serverSideOnly = false`
- New galaxy is strongly recommended for best results.
